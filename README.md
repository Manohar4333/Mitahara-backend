# Mitahara FoodOrder Backend

This project is the backend for the FoodOrder application, built with Node.js and Express.

## Project Structure

- `app.js`: Main application entry point
- `bin/www`: Server startup script
- `model/`: Data models (orders, product, users)
- `routes/`: API route handlers
- `views/`: Jade templates for rendering views
- `public/`: Static assets (images, stylesheets, javascripts)

## Setup

1. Clone the repository
2. Run `npm install` to install dependencies
3. Start the server with `npm start` or `node bin/www`

## Features

- User authentication
- Product management
- Order processing
